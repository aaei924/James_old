# DDark Wiki Engine
작명부터 구린 위키엔진.
the seed 를 보고 자체 엔진이 있으면 좋겠다 싶어서 제작을 시작한 엔진 이나 개같이 만듬

 - Frameworks : CodeIgniter
 - DBMS : Only MySQL (MariaDB)
 - Cache System : Redis (Redis better than Memcached)
 - Recommended Web Server : NGINX + PHP 7 or higher
